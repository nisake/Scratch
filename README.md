# Scratch ♥ けずって、あらわれる

> お気に入りの写真を、宝くじやスクラッチアート風にけずって楽しむWebアプリ。

**[▶ 公開ページはこちら](https://nisake.github.io/Scratch/)**

## ✨ できること

- 好きな画像をアップロードして、銀色のフィルムで覆う
- 指やマウスでこすって削ると、下から画像が現れる
- 4つのモードを楽しめる：
  - 🎟️ **宝くじ風（銀）** — クラシックなスクラッチカードの世界観
  - 🌟 **ゴールド** — リッチな金色フィルム
  - 💗 **ハートピンク** — 優しいピンクのフィルム
  - 🎨 **スクラッチアート（黒）** — 黒い表面を削ると下から色が現れる古典的アート遊び
- ブラシのふとさ・けずりやすさ（硬め／中／やわらか）を調整可能
- スクラッチアートモードでは下地の色も選べる（虹・コーラル・スカイブルーなど9種）
- 完成した作品はPNGで保存可能

## 🔒 プライバシー

画像はあなたのブラウザの中だけで処理されます。サーバーへのアップロードや保存は一切ありません。

## 🛠️ 技術

- Vanilla JavaScript + HTML5 Canvas
- ビルドステップなし（静的ホスティング）
- GitHub Pages でホスティング
- レスポンシブ対応（PC・スマホ・タブレット）
- タッチデバイス対応

## 🚀 ローカルで動かす

依存ライブラリゼロ。クローンしてブラウザで `index.html` を開くだけ。

```bash
git clone https://github.com/nisake/Scratch.git
cd Scratch
# 任意のローカルサーバーで起動（例）
python -m http.server 8000
# → http://localhost:8000 にアクセス
```
